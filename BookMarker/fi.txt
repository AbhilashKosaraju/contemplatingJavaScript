Pygame:
Go to the link, 
https://www.lfd.uci.edu/~gohlke/pythonlibs/
Search for the Pygame,
Download the appropriate .whl file
A WHL file is a package saved in the Wheel format, which is the standard built-package format used for Python distributions. It contains all the files for a Python install and metadata, which includes the version of the wheel implementation and specification used to package it. WHL files are compressed using Zip compression.

pygame-1.9.3-cp36-cp36m-win_amd64.whl for Python 3.6 (64 bit)

python -m ensurepip 

python -m ensurepip --upgrade

pip install wheel 
Additional Libraries will be downloaded.

pip install wheel --upgrade

Locate the .whl file we downloaded in command prompt
install  pygame-1.9.3-cp36-cp36m-win_amd64.whl

And the pygame has been installed successfully.


Note: Open command line as Administrator.
